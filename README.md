📌 Overview

The Bank Management System is a Python-based application developed using Object-Oriented Programming (OOP) principles to simulate real-world banking operations in a structured and efficient manner. The system enables users to perform essential banking activities such as deposits, withdrawals, and balance inquiries while ensuring data security and integrity.

This project focuses on implementing clean code practices and modular design by leveraging key OOP concepts, making the application scalable, maintainable, and easy to extend. It serves as a practical demonstration of how fundamental programming principles can be applied to develop real-world software solutions.

🚀 Key Features
Account creation and management
Secure deposit and withdrawal operations
Real-time balance inquiry
Interest calculation for savings accounts
User-friendly command-line interface
🧠 OOP Concepts Implemented
Encapsulation: Protects sensitive data such as account balance
Inheritance: Enables reuse of common functionality
Polymorphism: Supports method overriding and flexibility
Abstraction: Simplifies complex operations for the end user
🛠️ Technology Stack
Language: Python
Programming Paradigm: Object-Oriented Programming (OOP)
